SECURITY MEASURES 
Personnel Rules 
Access to personal data is limited to the LiveLike backend development team, consisting of four engineers and 
managers. Stringent security measures are implemented for all personnel with access to personal data, including: 
1.Multi-Factor Authentication (MFA) for all services including Google, Bitbucket, and AWS. 
2.Identity and Access Management (IAM) for AWS access., with restricted root access for all users. 
3.All work devices (laptops, tablets and phones) are password-protected and encrypted. 
Cloud Security 
We use AWS Guard Duty , AWS’s intelligent threat detection service, for continuous monitoring of our cloud 
environment. 
GuardDuty analyzes AWS CloudTrail events, VPC Flow Logs, and DNS logs using machine learning and threat 
intelligence to identify unauthorized access, data exﬁltration attempts, cryptomining, or malware activity. Findings 
are integrated with our alerting channels (Slack, Email, SMS) for immediate action. 
We also leverage GuardDuty’s reporting and integration with AWS Security Hub to support compliance monitoring, 
including periodic exports for GDPR and regulatory audits .
